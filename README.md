# Jukebox Backend Assignment

REST API with single GET endpoint which returns a list of jukeboxes based on setting id.
The GET endpoint can filter by desired jukebox model name.

## Start application

1. `npm install`
2. `npm run start`
3. Access to Postman collection in `docs/Jukebox.postman_collection.json` to ping GET endpoint.
