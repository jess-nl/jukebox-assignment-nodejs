# Jukebox Backend Assignment

REST API with single GET endpoint which returns a list of jukeboxes based on setting id.
The GET endpoint can filter by desired jukebox model name.

## Start application

1. Ensure the **.env** variables are provided
2. `npm install`
3. `npm run start`
4. Access to **Postman** collection in `docs/Jukebox.postman_collection.json` to ping GET endpoint
